Real development steps:

1. Create a virutal environment 							 'python3 -m venv env'
2. activating virtual environment 							 'source env/bin/activate'
3. Installing django 										 'pip install django'
4. Creating a project 										 'django-admin startproject judge'
5. make migrations 											 'python manage.py makemigrations'
6. migrate													 'python manage.py migrate'
7. create an app
8. make static and templates folder.
9. initialising the git repo.
10. start coding.
